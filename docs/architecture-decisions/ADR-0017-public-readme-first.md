# ADR-0017: Use README as the Portfolio Entry Point

## Status

Accepted

## Context

Recruiters and interviewers are likely to open the repository root first.

## Decision

Keep `README.md` as the primary portfolio entry point. It should summarize project identity, scope, current baseline, document links, and boundaries.

## Consequences

- The repository is easier to review quickly.
- Detailed docs remain available for deeper inspection.
- README must be updated whenever major backend or infrastructure code is imported.
