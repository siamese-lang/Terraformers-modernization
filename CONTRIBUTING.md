# Contributing Guardrails

This repository is a public portfolio repository for Terraformers modernization work.

Before adding code or documentation, check:

- No AWS access key, secret key, session token, DB password, GitHub token, kubeconfig, tfstate, or real tfvars is included.
- No team member's work is presented as solo work.
- Frontend and AI-generation quality work do not become the main topic.
- Backend and cloud infrastructure modernization remains the central theme.
- Any evidence is sanitized before commit.

Recommended commit scope:

- backend modernization
- RDB/Flyway schema
- runtime config and secret contract
- Terraform infrastructure
- CI/CD and validation
- runbook and evidence templates
